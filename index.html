What is Prometheus?


Prometheus is an open-source system for monitoring services and alerts based on a time series data model. Every DevOps professional should be familiar with the Prometheus tool. It can help to records real-time metrics in a time series database built using an HTTP pull model, with flexible queries and real-time alerting. It is a free software application used for event monitoring and alerting. This tool provides a key component for a modern DevOps workflow: keeping watch over cloud-native applications and infrastructure, including another popular CNCF project, Kubernetes.
